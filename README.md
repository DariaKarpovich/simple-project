## <p id="part2" align = "center"> Syntax MarkDown </p>

**Синтаксис MarkDown**:
- Basic Syntax;
- Extended Syntax.

---

| Element | Markdown Syntax |
| ----------- | ----------- |
| ***Basic Syntax***| 
| Heading | `# H1 ## H2 ### H3`|
| Bold | `**bold text**`|
| Italic | `*italicized text*`|
| ***Extended Syntax***|
| Strikethrough | `~~The world is flat.~~`|
| HeadingID | `### My Great Heading {#custom-id}`|


*Информацию я взяла с сайта [https://www.markdownguide.org/](https://www.markdownguide.org/).*
